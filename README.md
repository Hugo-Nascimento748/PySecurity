# 🛡️ PySecurity - Gerenciador de Senhas Seguro

O **PySecurity** é uma aplicação desktop desenvolvida em Python para o gerenciamento seguro de credenciais. O projeto foca em aplicar conceitos reais de **Segurança da Informação** e **Desenvolvimento de Interfaces Modernas**, garantindo que senhas sejam armazenadas de forma criptografada e protegida.

---

## 🚀 Funcionalidades

- **Criptografia AES (Fernet):** Utiliza criptografia simétrica de nível militar para transformar senhas em tokens ilegíveis.
- **Interface Moderna (GUI):** Desenvolvida com a biblioteca `CustomTkinter`, oferecendo uma experiência visual limpa com suporte nativo a Dark Mode.
- **Gestão de Dados (CRUD):** Permite salvar, listar e excluir credenciais de forma intuitiva.
- **Arquitetura Modular:** Separação clara entre a lógica de criptografia/dados (`engine.py`) e a interface visual (`main.py`).

---

## 🛠️ Tecnologias Utilizadas

- **[Python](https://www.python.org/):** Linguagem principal.
- **[CustomTkinter](https://github.com/TomSchimansky/CustomTkinter):** Interface gráfica moderna e responsiva.
- **[Cryptography](https://cryptography.io/):** Biblioteca padrão para implementação de segurança e cifras.
